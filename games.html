Стікмен

Гра "Стікмен" - це захоплююча платформер гра, де ви керуєте персонажем-паличкою, який повинен подолати різні перешкоди та вирішувати математичні завдання, щоб дістатися до виходу. Використовуйте стрілки для переміщення та пробіл для стрибків.

✓ 10 рівнів складності
✓ Математичні завдання
✓ Рухомі платформи
✓ Анімований персонаж

Блоковий світ

"Блоковий світ" - це 3D гра у стилі Minecraft з математичними елементами. Ви можете будувати та руйнувати блоки, кожен з яких містить математичні рівняння або задачі. Гра має два режими: творчий (з можливістю літати) та режим виживання.

✓ Відкритий 3D світ
✓ Математичні блоки
✓ Два ігрових режими
✓ Збереження карти

✓ 4 теми питань

✓ Анімації та звуки
